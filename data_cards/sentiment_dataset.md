# Dataset Card: Sentiment Dataset

**Summary:** A small synthetic dataset containing short product review
sentences labeled as `positive` or `negative`.  Intended for
demonstration and testing of sentiment analysis models.

**Number of records:** 6

**Fields:**

- **text**: The review text string.
- **label**: The sentiment label (`positive` or `negative`).

**Source:** Generated manually for example purposes.  No real user
data is included.

**Purpose:** This dataset is designed to illustrate how to structure
dataset files and data cards.  It may be used to build toy sentiment
classifiers.

**Ethical Considerations:** Because the data is synthetic and contains
no personally identifiable information, there are minimal privacy
concerns.  However, models trained on this small dataset will not
generalize to real‑world sentiment analysis tasks.
